# 窝囊费计算器 🐷

一个有点好笑、但认真工作的 Android App：输入月薪或年薪后，按照工作日规则和上下班时间，把收入换算成“每秒赚多少钱”，并在首页实时显示“今天已经赚到”的金额。

核心乐趣很简单：哪怕每秒只涨一点点，也要亲眼看到它在涨。

## 功能亮点 ✅

- 支持月薪 / 年薪输入
- 实时显示今天已经赚到的钱
- 自动计算每秒收入、每小时收入
- 根据用户设置的上下班时间计算工作时长
- 低金额场景下自动提高小数位精度，避免一直显示 `¥0.00`
- 支持中国法定工作日和每周固定工作日两种规则
- 内置 2026 年中国法定节假日与调休数据
- 支持联网同步当年法定节假日数据，并缓存到本地
- 打开 App 时优先读取当年缓存数据，没有缓存时再联网同步
- 支持浅色 / 深色 / 跟随系统外观模式
- 设置页修改后立即生效，不需要手动保存
- 关于页内置版本号、隐私说明、GitHub 和反馈入口
- 使用 DataStore 保存本地设置
- 核心计算、工作日规则、节假日解析和设置映射均有单元测试

## 界面与素材 🎨

v0.4.0 开始，界面改用手绘风 PNG 素材，不再用临时 Canvas/SVG 仿画。当前素材包括：

- 打工人、下班休息人
- 小猪、存钱罐、金币
- 钱包、时钟、日历、调色盘
- 隐私盾牌、反馈气泡、设置齿轮

原始元素集保存在 `design-assets/element-set.png`，抠出的透明素材保存在 `design-assets/cutouts/`，App 内使用的资源在 `app/src/main/res/drawable-nodpi/`。

## 当前规则说明 📅

中国法定工作日模式下：

- 优先使用当前系统年份对应的本地缓存数据
- 如果没有缓存，则尝试联网同步当年法定节假日数据
- 同步成功后会保存到本地，后续打开 App 可直接读取
- 如果同步失败且当前年份有内置数据，则使用内置数据
- 如果既没有缓存也没有内置数据，则回退到固定周一到周五
- 不会把上一年的法定日历误用于下一年

## 项目结构 🗂️

主要 UI 代码按功能组织：

- `ui/app`：App 编排、页面切换、系统栏、外链、主题模式解析
- `ui/home`：主屏收入展示
- `ui/settings`：设置页整体结构和设置模型
- `ui/settings/sections`：薪资、工作时间、工作日、外观等设置区块
- `ui/settings/components`：设置页复用控件
- `ui/about`：关于页面
- `ui/common`：格式化、文案标签、UI 常量、通用素材组件
- `ui/sync`：法定日历同步状态逻辑
- `ui/theme`：Material 主题

## 隐私 🔒

薪资、上下班时间、工作日规则和外观设置都保存在本机，不会上传。

App 仅在同步中国法定节假日数据时使用网络。

详细说明见 [PRIVACY.md](PRIVACY.md)。
